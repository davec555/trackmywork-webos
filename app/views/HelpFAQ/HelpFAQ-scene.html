Frequently Asked Questions

  1. What is the difference between the free and the full version?

    Here is a list of differences:

    • Graphical rendering of statistics
    • Automatic notifications for check in and out
    • Dashboard mode for easier check in and out
    • Automatic calculated breaks

  2. I have entered data for some category, but now it can't see it in the Weekview. In the day view the data is visible. Where is it?

    Thats easy. Simply add the category of the missing entries to the category filter. Then the data will be shown up and calculated in the week view.

  3. How can I get TMW being placed in the dashboard all the time?

    Simply use the back-gesture in the week view. The app gets minimized to the dashboard then. From there you can easily check in and out, without opening the app.

  4. Will there be support for automatic check in/out using GPS or Wifi?

    No. I checked the reliability of GPS and WiFi events and was not satisfied. It seemed to be more work, correcting the automatic generated entries, then checking in and out manually.
    The dashboard mode gives the user support to check in and out as easy as possible.

  5. Can I export TrackMyWork-data in a spreadsheet?

    Yes. Currently it is not comfortable, but possible. Do it in the following steps:

    1. Export the data in a mail, to be sent by yourself
    2. Extract the data, you want from your received email by copy and paste in a file called like YourSheetName.csv
    3. Import this file into your spreadsheet software
    4. You're done!
